# News Management System Backend
A production-grade FastAPI backend for managing news articles with image & video upload support, analytics, and mobile app integration (Android Studio Compatible).

## Features
- Create, Read, Update and Delete operations for news posts
- Firebase Firestore Database
- Firebase Storage for image/video uploads
- SEO-friendly slugs
- Likes counter
- Ananlytics dashboard API
- Ready for Android mobile integration
- Fully deployable on Render

## Tech Stack
- Python FastAPI
